# Wedding Expense Tracker

A **Next.js web application** to help plan and manage their wedding finances.  
Track your **budget, expenses, and savings** by category, and visualize progress with tables and progress bars.

---

## ✨ Features

- **Budget Planner**
  Allocate funds to categories (venue, catering, decor, etc.).

- **Expense Tracking**  
  Log expenses and monitor spending in real time.

- **Savings**
  Record savings and compare against planned budget.

- **Dashboard**
  - Category‑wise breakdown of Budget, Spent, Saved, Remaining.
  - Visual **progress bars** for each category.
  - Easy overview of financial health.

- **Responsive UI**  
  Elegant design with decorative backgrounds and smooth layout.

---

## 🛠️ Tech Stack

- **Frontend**: Next.js (App Router), React, TypeScript
- **Styling**: Tailwind CSS
- **Data**: JSON / API routes for budget, expenses, savings
- **Version Control**: Git + GitHub

---

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
